Master Skills

Core routing and orchestration skills that coordinate other skills.

Master

pi-engineer

Master router that dispatches to the right specialist skill based on your request.

/skill:pi-engineer
Triggers: "set up pi", "improve my harness", "audit my setup"
Composed

pi-init

Bootstrap a complete pi harness from scratch. Orchestrates scaffold, permissions, memory, and hooks.

/skill:pi-init
Triggers: "initialize harness", "bootstrap this project"
Composed

pi-loop

Continuous improvement cycle: audit, identify weakest pillar, improve, validate, repeat.

/skill:pi-loop
Triggers: "improve my harness", "keep improving"
Composed

pi-audit

Evaluate harness maturity across all 7 pillars. Scores 0-21 with detailed recommendations.

/skill:pi-audit
Triggers: "audit my harness", "check my setup"

Using Skills

Invoking Skills
// Direct skill invocation
/skill:pi-audit

// Skill with arguments
/skill:pi-scaffold set up for a React TypeScript project

// Chain skills
/skill:pi-init bootstrap my project, then run pi-audit
Skill Composition
// pi-init orchestrates multiple skills:
pi-init = pi-scaffold + pi-permissions + pi-memory + pi-hooks

// pi-loop orchestrates audit + improvement:
pi-loop = pi-audit + weakest-pillar-skill + validate

// Custom composed skill:
my-deploy = pi-scaffold(build) + pi-gates(test) + pi-routing(push)

MCP Extensions

Pi supports MCP (Model Context Protocol) via extensions. Add MCP servers to extend pi with browser automation, file system access, and more.

MCP

Playwright MCP

Browser automation via Playwright MCP. Navigate pages, extract content, fill forms, take screenshots.

.pi/extensions/playwright-mcp.ts
Tools: browser_navigate, browser_screenshot, browser_click, browser_fill, browser_evaluate
MCP

Custom MCP

Add any MCP server as a pi extension. Connect to code generation, search, or custom tools.

.pi/extensions/*.ts
Pattern: Register MCP tools via ExtensionAPI
Available Playwright Tools
// Browser automation tools registered as:
/skill:playwright_browser_navigate     // Navigate to URL
/skill:playwright_browser_screenshot  // Take page screenshot
/skill:playwright_browser_click       // Click element
/skill:playwright_browser_fill        // Fill form field
/skill:playwright_browser_evaluate    // Run JavaScript
/skill:playwright_browser_snapshot    // Get accessibility tree
